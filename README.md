# SMHTravel

## Prerequisites

 - python 3.x
 - pip

Or preferably, have Anaconda/Miniconda setup.

## Setup

```bash
pip install -r requirements.txt
```
or use `conda`.

## Run

```bash
uvicorn app:app --host localhost --port 8000
```

Access the app via a browser at the mentioned `--port` value, at localhost.
